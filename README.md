# Food-waste-rescue-radar
An AI-powered web app to predict and reduce food waste for schools and community events.

## Features
- Predict waste rate
- Waste reduction guide

## How it works
Users enter:
- Event type:
  - School lunch
  - Community event
- Location:
  - Cafeteria
  - Party / Event venue
- Expected attendance
- Actual attendance
- Food prepared
- Leftover portions
- Meal type
- Weather conditions
- Notes
   
The system calculates waste risk and suggest practical action.

## Technology
- Python
- Streamlit
- Github

## Responsible AI
The AI provides suggestions only.

Food safety must be inspected by humans. 

The AI does not decide whether leftovers are safe to be donated or continue serving.
